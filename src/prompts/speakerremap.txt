これらは議事録のファイルです。全体の話の流れを確認し、話者(speaker)を推測し、話者名の変換マップをJSON形式で出力してください：
# マッピング例:
- 「話し手1」→「山谷」
- 「人物A」→「星山」

# 同じ実在人物が複数の異なる話者表記で登場することがあります。
例えば「話し手1」も「人物A」も同じ「山谷」さんである可能性があります
会話の文脈から同一人物だと判断できる場合は、同じ名前にマッピングしてください
# 話者の名前がわからなかった場合は、不明やunknownとはせずに、識別できる名前にすること

# フォーマット：

```json
{
  "話し手1": "山谷",
  "人物A": "山谷",
  "人物B": "話者A",
}
```

このマッピングJSONは後続の処理でファイル変換に使用されます。
